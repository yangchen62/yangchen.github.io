eslint-js

Posted on 2020-12-14

  • 定义变量：使用es6的语法，不使用varprefer-const, no-const-assign,no-var

  • 不要使用new新建对象

  • 不确定对象的key值的时候使用计算变量

  • 使用shorthand方式给对象方法赋值object-shorthand

  • shorthand命名方式在对象的前面

  • 对象key值尽量不用字符串quote-props

  • 建议使用扩展运算符prefer-object-spread

  • 不要使用array的构造函数 no-array-constructor

  • 使用扩展运算符复制数组

  • 把类数组转换为数组

  • 多行定义变量

  • 合理运用结构赋值

  • 字符串用单引号

  • 模板字符串使用前后没有空格

eslint-react

Posted on 2020-12-14

Class vs React.createClass vs stateless

  • 使用class extends React.Component : react/prefer-es6-class react/prefer-stateless-function

    没有state、ref的纯组件:

Mixins

  • 不要使用Mixins

命名

  • 文件名: 大驼峰命名组件文件。

  • 命名规范

    1. handleXxx 处理数据
    2. getXxx 获取数据或者组件
    3. onXxx 处理事件
    4. className命名使用cm-

  • 引用: react组件使用大驼峰命名，组件的实例使用小驼峰命名. eslint: react/jsx-pascal-case

  • 高阶组件的命名形式

  • Props命名

定义

  • 在定义class的时候直接定义和class的名字，而不是使用display

对齐方式

  • JSX的书写方式. eslint: react/jsx-closing-bracket-location react/jsx-closing-tag-location

单双引号

  • JSX的属性使用双引号，对象的值使用单引号

空格

js和jsx都使用四个空格而不是使用tab

  • 自闭和标签闭合标签前有一个空格. eslint: no-multi-spaces, react/jsx-tag-spacing

  • JSX{}前后没有空格. eslint: react/jsx-curly-spacing

Props

  • 普通的props使用小驼峰，组件使用大驼峰.

  • props的值是布尔值时请省略不写. eslint: react/jsx-boolean-value

  • img的alt标签要一直存在，不能不写. eslint: jsx-a11y/alt-text

  • 不要再dom中使用 accessKey . eslint: jsx-a11y/no-access-key

  • 建议不要使用index作为key. eslint: react/no-array-index-key

  • 设置默认props

括号

  • JSX跨越多行返回的时候用（）括起来. eslint: react/jsx-wrap-multilines

标签

  • 没有子节点的时候使用单标签 eslint: react/self-closing-comp

  • 有多个属性的时候，闭合标签换行. eslint: react/jsx-closing-bracket-location

方法

  • 方法的前缀不要使用下划线

排序

  • 每个react组件的排序按照下面的顺序:
  1. optional static methods
  2. constructor
  3. getChildContext
  4. componentWillMount
  5. componentDidMount
  6. componentWillReceiveProps
  7. shouldComponentUpdate
  8. componentWillUpdate
  9. componentDidUpdate
  10. componentWillUnmount
  11. event handlers starting with ‘handle’ like handleSubmit() or handleChangeDescription()
  12. event handlers starting with ‘on’ like onClickSubmit() or onChangeDescription()
  13. getter methods for render like getSelectReason() or getFooterContent()
  14. optional render methods like renderNavigation() or renderProfilePicture()
  15. render

  • 如何定义 propTypes, defaultProps, contextTypes, 等等
